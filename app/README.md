# DailyVideoOps Web App

## Features
- User signup/login
- Per-user posts
- Multi-platform targeting (Facebook, Instagram, WhatsApp, YouTube, TikTok)
- Scheduled queue with simulated posting logs

## Run
```powershell
cd app
npm.cmd install
npm.cmd start
```

Open `http://localhost:3000`.

## Data
Stored in `app/data/db.json`.

## Important
This MVP simulates posting. Real social posting requires OAuth + official APIs per platform.
